# 🎲 Flutter Dice Rolling

A simple and interactive Flutter application that simulates rolling a dice. The app generates a random number from **1 to 6** whenever the user presses the roll button, accompanied by a smooth animation and realistic dice rolling sound effects.

## ✨ Features

* 🎲 Single dice rolling interface
* 🔢 Generates random numbers from **1 to 6**
* 🎞️ Smooth dice roll animation
* 🔊 Dice rolling sound effect
* 📱 Clean and responsive UI
* ⚡ Built with Flutter

## 🚀 Getting Started

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* An Android emulator or physical device

### Installation

1. Clone the repository

```bash
git clone https://github.com/vg3371396-bit/Flutter-Dice-Rolling.git
```

2. Navigate to the project directory

```bash
cd dice
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```

## 🎮 How It Works

1. The app displays a dice and a roll button.
2. When the user taps the button:

   * A dice rolling sound is played.
   * The dice animates briefly.
   * A random number between **1 and 6** is generated.
   * The corresponding dice face is displayed.
